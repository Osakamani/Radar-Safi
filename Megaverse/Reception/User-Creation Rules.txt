For every 1 Organic Farmer the system supports 2 Development Students. Subsequently, admitting a student Programmer hence requires that the student programmer must have essential farming skills. Two student programmers can deliver better than one because their farming activities can support an intensive system development and programming internship student who is going to be trained to head the software project.

Field Agents to be hired according to cultural regions so that all cultures are represented. System is highly encouraged to hire candidates with multiple capabilities and interests.

Given that the number of field agents is fixed, that number can be used as a basis for determination of the orgware capacity of Radar Safi Core systems.
